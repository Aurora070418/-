# **App Name**: OurJourney

## Core Features:

- Couple Pairing & Profiles: Allow partners to create accounts and securely link them, providing a personalized and shared experience.
- Shared Wishlist: A private, shared list where each partner can add gift ideas they'd like, visible to the other for special occasions.
- Mutual Expense Log: A simple interface for partners to daily record and view their combined expenditures, promoting transparency.
- Joint Activity Planner: Create and manage a list of activities, goals, or experiences the couple wants to do together.
- Special Date Countdown: Record and receive countdowns for important anniversaries, birthdays, and other significant couple milestones.
- Days Together Tracker: Prominently display the total number of days the couple has been together since their relationship start date.
- AI Date Idea Generator Tool: A tool that generates personalized and creative date ideas based on preferences such as mood, budget, or activity type.

## Style Guidelines:

- Primary color: A warm and inviting soft rose shade (#CC6688) to represent connection and affection. This is complemented by a light color scheme.
- Background color: A very light, desaturated variant of the primary hue (#F4F0F2), providing a clean and soft canvas for content.
- Accent color: A deep, elegant violet (#BF26BB), chosen to add sophistication and visual interest while remaining analogous to the primary color.
- Body and headline font: 'PT Sans', a humanist sans-serif chosen for its clear readability, modern look, and touch of warmth suitable for personal content.
- Utilize modern, clean outline icons for core functionalities and navigation, maintaining a simple yet elegant aesthetic.
- A clean, spacious layout with a mobile-first approach, featuring clear content hierarchy and dedicated sections for each feature for easy information digestion.
- Implement subtle, fluid animations for state changes and user interactions (e.g., button hovers, content loading), enhancing the user experience without distraction.
